# SeqDesign
Supplementary code for Group Sequential Design with Posterior and Posterior Predictive Probabilities manuscript

There are seven code files for this manuscript.

- 01_dynamic_functions: details the functions used in Section 6.1 of the main text
- 02_dynamic_results: reproduces the results in Section 6.1 of the main text
- 03_ex1_functions: details the functions used in Appendix D.1 of the supplement
- 04_ex1_results: reproduces the results in Appendix D.1 of the supplement
- 05_ex1_appd2: reproduces the results in Appendix D.2 of the supplement
- 06_ex2_functions: details the functions used in Appendix E of the supplement
- 07_ex2_results: reproduces the results in Appendix E of the supplement

There is also the "jags_surv.txt" file that is used to implement JAGS for the survival regression model in
Appendix D; the "jags_beta.txt" file is used to implement JAGS for the beta model in Appendix E.
